Introduction

MH2 Software System Requirements.

Minimum Basic Requirements to Run MH2

OS:

  • Windows 10 or 11 (64-bit) (.exe available)
  • Linux, Debian 12 version. Example: Ubuntu 24.04 nickname: “Noble Numbat” or later (.deb available)
  • Linux, Red Hat 9.x version. (rpm not yet available)
  • Mac OS X 10.11 El Capitan

Processor (CPU):

  • Intel Core i5-4570
  • AMD Ryzen 5 1500X

Memory (RAM): 8 GB.

Storage: 5 GB available HDD or SSD space. More space may be needed for CC & Downloads

Graphics (GPU):

  • NVIDIA GeForce GTX 770
  • AMD Radeon R9 280X
  • Intel HD 530

Network: Broadband internet connection.

Optional Requirements

Newest or long term version of Blender installed and MH2 Socket.
