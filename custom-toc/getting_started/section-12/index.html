Windows Installation

Your guide to install on a Windows 10 and Windows 11 operating systems.

Introduction

It has never been easier to participate in Makehuman 2! There is a ready to use download available here:
Windows 10 & 11 Download

Section 1-Check your version!

Standard Windows installation for win 10 & 11 os.

Section 1.1-Installation

Once you download the exe from this link you can simply double click it. You will likely be instructed,
the application is a potential threat. This message can be dismissed.

Once you do so, you will be greeted by the wise installation wizard-who will guide you through the
installation procedure. You can choose a custom directory-however for alpha it is advised to use
the default installation directories.
Once the application is installed you will be promted of success of istallation-and an icon should then
be placed upon your desktop.

Section 1.2-First time Running

Note that during the first running, or any time you delete your conf file in %appdata% you will be instructed
to point to the custom content directory of choice.

Section 2-Uninstall

If for some reason, you need to remove the program, there is an uninstall exe included in the install directory
uninstall.exe. You can also use the system uninstall programs panel in windows system apps & programs menu.
